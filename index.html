String Operators

The addition operator + can be used to concatenate strings and += operator can be used to assign and concatenate a string.

For Example: var greeting = "Hello" + "Alby Baby"; =====>The value in variable greeting is "Hello Alby Baby" greeting += " Have a nice day!!"; ======>Now the value changes to "Hello Alby Baby Have a nice day!!"

When used on strings, the + operator is called the concatenation operator.

Adding Strings and Numbers

One of the fascinating thing about Javascript is you can add numers and strings .If you add number to a string JS converts the numbers to strings

Example: var y = "8" + 8; =======> Value of y = 88 var steps = "step" + 1; ========>Steps = step1

Example: var x = 50; if(x>=30){ alert("You pass the benchmark");} if(username==="alby"){alert("Welcome Alby!");}

In javascript,comparing differnt types may be a little different.It doesn't throw an error. When comparing a string with a number js will convert the number to a string at the time of comparison.If we are comparing with an empty string it returns 0.while with a non numeric string it converts to NaN which is always false.This NaN has many uses in js

Example: 2 > "12" ========> false "2" >"12"=========>true (because both are strings and alphabetically 2 is greater than 1) 34 =="alby"=========>false(because "alby" is converted to NaN)
